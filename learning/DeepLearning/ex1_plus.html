z-score标准化¶

z-score标准化：适用于属性的最大值和最小值未知，或有超出取值范围的离群数据的情况。

标准差：数据集和平均值的差异大小

$$z=\frac{x-\bar{x}}{σ}$$
  1. data - data.mean()
    • 新数据的平均值 = 0
    • 下移
  2. / data.std()
    • 新数据的标准差 = 1
    • 压扁
    • 一个数据，距离平均数多少个标准差

我们不需要把这一步应用到x0中，因为x0总是等于1的，所以它不可能有为0的的平均值

提出模型¶

代价函数¶

同ex1，此函数参数个数是可变的

代价函数：$$ J\left( \theta \right)=\frac{1}{2m}\sum\limits_{i=1}^{m}{\left( {h_\theta}\left( {x^{(i)}} \right)-{y^{(i)}}\right)}^2 $$

其中假设函数：$$ {h_\theta}\left(x \right)={\theta^T}X={\theta_0}+{\theta_1}{x_1} + ... +{\theta_i}{x_i} $$

In [99]:
def computeCost(x, y, theta):
    inner = np.power(x * theta.T - y, 2)
    return np.sum(inner)/(len(x)*2)

梯度下降函数¶

$${\theta_j}:={\theta_j}-\alpha \frac{\partial }{\partial {\theta_j}}J\left( \theta \right)$$
In [103]:
def gradientDescent(x, y, theta, alpha, iters):
    cost = np.zeros(iters)
    
    for i in range(iters):
        dtheta = (x * theta.T - y).T * x / len(x)
        theta = theta - alpha * sum(dtheta)
        
        cost[i] = computeCost(x, y, theta)
        
    return theta,cost
